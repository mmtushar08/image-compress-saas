# Deployment Scripts

This directory contains the optimized deployment script for the Shrinkix project.

## Files
- `deploy_optimized.js`: The main secure deployment script using environment variables.
- `nginx.conf`: Nginx configuration for the VPS.
- `DEPLOYMENT.md`: Documentation for deployment.

## Usage
See [SECURE_DEPLOY.md](../../SECURE_DEPLOY.md) for instructions.

## Security Note
Previous legacy scripts containing hardcoded credentials have been removed for security.
