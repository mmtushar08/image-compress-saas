Automated Image Optimization & WebP Conversion powered by Shrinkix API.

== Description ==

Shrinkix Image Optimizer automatically compresses your images and generates WebP versions to speed up your WordPress site. 

**Features:**
*   **Auto-Compression**: Optimize images on upload automatically.
*   **WebP Generation**: Create next-gen WebP versions of your images.
*   **Background Processing**: Uses background processing to prevent site slowdowns.
*   **Bulk Optimization**: Process your existing media library with one click.
*   **Backup**: Option to keep original full-resolution images.
*   **SaaS Powered**: Offload the heavy lifting to our blazing fast API.

== Installation ==

1. Upload the `shrinkix-wp` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure your API Key in Settings > Shrinkix.

== Frequently Asked Questions ==

= Do I need an API Key? =
Yes, you need a Shrinkix API key. Sign up at Shrinkix.com (example) to get one.

= Does it replace my original images? =
Yes, it optimizes the file in place. You can enable "Backup Originals" in settings to keep a copy.
